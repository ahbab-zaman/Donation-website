Frequently Asked Questions

What is the Document Object Model (DOM)?

DOM stands for (Document Object Model). Document -> The Whole webpage, Object -> Every single element in the webpage, Model -> The structure of the webpage that connects all the elements. Basically, It allows the codes to access dynamically and change the contents and it's styles related to it's document.

How do you select an element from the DOM?

A document can be selected in many ways. Steps are below :

1. It can be accessed by it's unique ID (getElementById('name of the specific ID')). 2. It can be accessed by it's class name(getElementsByClassName('name of the class name')). 3. It can be accessed by it's tag name(getElementsByTagName('name of the specific tag name')).
What is event delegation in the context of the DOM, and why is it useful?

Event Delegation refers to handle the root element of an object instead of handle every child element. It is useful because it is efficiently manageable and handle the child elements by attaching a single event listener to the parent element.

How do you manipulate an element's attributes and styles using the DOM?

If anyone wants to remove or add several classes form an html element then javascript provide a method named classList(). It's usage is to remove or add a class in the element. Besides, if anyone wants to set,get or remove a href, id or src from the html element then use setAttribute(),getAttribute(), removeAttribute() method respectively. Moreover, to change styles in an element, at first access the element by it's className, unique ID or tagName and then set the desired styles in it using the style property directly.
